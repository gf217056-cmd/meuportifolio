# projeto Aula de Github

Este é um projeto simples criado para aprender:

- como usar o Github
- como criar um site básico
- como organizar arquivos (HTML e CSS)

## arquivos do projeto

- index.html → estrutura do site
- style. CSS → estilo (cores, layout)

  ## objetivo

  criar uma pagina simples com titulo, texto estilo.
